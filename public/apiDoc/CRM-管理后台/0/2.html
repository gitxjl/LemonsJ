接口描述

Headers

参数名称 是否必须 类型 默认值 描述
Authorization 是 text Bearer {{token}}
Content-Type 是 text application/x-www-form-urlencoded

Params GET 请求参数

参数名称 是否必须 类型 默认值 描述
XDEBUG_SESSION_START 是 1565744662580
per_page 是 10
page 是 1
sn 是 订单
type 是 类型
charge 是 1 收费类型 1 => '收费', 2 => '收费'
pay_way 是 支付方式 1 => '支付宝', 2 => '微信'
agentcode 是 DL00350 ID
username 是 络 名称

Body POST 请求参数

Boby类型：urlencoded

参数名称 是否必须 类型 默认值 描述
x-www1 是 text x-www1 x-www1
x-www2 否 text x-www2 x-www2
